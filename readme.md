## PWD 
La commande **PWD** permet de déterminer ou est ce qu'on se trouve dans son arborescence de repertoire.  

>[!NOTE]
>
>- PWD : **P**rint **W**orking **D**irectory.  

- **Syntaxe :**

```shell
pwd 
```
